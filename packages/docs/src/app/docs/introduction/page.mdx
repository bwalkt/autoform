# Introduction

Autoform UI is a composable, token-first component library built on Base UI.
It is optimized for product teams building dense workflows where clarity matters
more than decoration.

<Callout variant="info">
  <CalloutText>
    Start with tokens and layout primitives. The rest of the system builds on the
    same semantic vocabulary.
  </CalloutText>
</Callout>
